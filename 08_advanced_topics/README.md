# Session 08 — Advanced Topics

## Learning Objectives

After this session you will be able to:

- Implement a Variational Autoencoder (VAE) and understand the ELBO objective
- Build a simple GAN and describe the training dynamics
- Export a trained model to ONNX and run inference in a standalone script
- Apply basic techniques for model compression (quantisation, pruning)
