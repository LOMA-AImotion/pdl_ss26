# utils — Shared Helpers

This folder contains utility modules shared across multiple sessions.

| Module | Description |
|--------|-------------|
| [`plotting.py`](plotting.py) | Common plotting helpers (training curves, confusion matrix, …) |
| [`training.py`](training.py) | Reusable training and evaluation loops |
| [`data.py`](data.py) | Dataset download helpers and custom `Dataset` classes |
