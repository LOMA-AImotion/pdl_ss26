# Session 02 — PyTorch Basics

## Learning Objectives

After this session you will be able to:

- Understand PyTorch's tensor operations and broadcasting rules
- Explain what autograd is and how backpropagation works under the hood
- Build and visualise a simple computation graph
- Run tensor operations on a GPU with `.to(device)`
