# Session 05 — Convolutional Neural Networks

## Learning Objectives

After this session you will be able to:

- Explain the convolution operation and understand receptive fields
- Build CNN architectures with `nn.Conv2d`, pooling, and skip connections
- Implement and train a ResNet-style network on CIFAR-10
- Visualise learned filters and feature maps
