# Session 07 — Sequence Models

## Learning Objectives

After this session you will be able to:

- Describe the vanishing gradient problem in vanilla RNNs
- Implement LSTM and GRU cells and understand their gating mechanisms
- Train a character-level language model
- Understand the self-attention mechanism and the Transformer encoder
