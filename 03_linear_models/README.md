# Session 03 — Linear Models

## Learning Objectives

After this session you will be able to:

- Implement linear regression with PyTorch from scratch
- Implement logistic regression for binary and multi-class classification
- Use `torch.nn.Module` and `torch.optim` to simplify model definition and training
- Understand and apply common loss functions (MSE, Cross-Entropy)

## Notebooks

| Notebook | Description |

| [`03_multiclass_classification.ipynb`](03_multiclass_classification.ipynb) | Softmax, Cross-Entropy, and the training loop |
