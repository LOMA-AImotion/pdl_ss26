# Session 04 — Neural Networks

## Learning Objectives

After this session you will be able to:

- Build multi-layer perceptrons (MLPs) with `nn.Sequential` and custom `nn.Module` classes
- Choose and apply appropriate activation functions (ReLU, GELU, Sigmoid, Tanh)
- Stabilise training with Batch Normalisation and Dropout
- Apply common weight initialisation strategies
