# Session 06 — Transfer Learning

## Learning Objectives

After this session you will be able to:

- Load and use pretrained models from `torchvision.models`
- Apply feature extraction (frozen backbone) and fine-tuning strategies
- Adapt a pretrained classifier to a custom dataset
- Understand domain adaptation and when transfer learning helps

## Notebooks

| Notebook | Description |
