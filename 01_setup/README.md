# Session 01 — Setup & Python Refresher

## Learning Objectives

After this session you will be able to:

- Set up a working Python / PyTorch environment (conda or pip)
- Recall key NumPy operations used throughout the course
- Create, manipulate, and inspect PyTorch tensors
- Move tensors between CPU and GPU
